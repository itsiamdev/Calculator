# Simple Calculator

This project is a simple calculator built using HTML, CSS, and JavaScript. It supports basic arithmetic operations (addition, subtraction, multiplication, division), square root calculation, and backspace functionality. The calculator features a clean, user-friendly interface with responsive design and smooth animations.

## Features
- Basic arithmetic operations (+, -, *, /)
- Square root calculation (√)
- Backspace (⌫) to delete the last character
- Clear (C) button to reset the display
- Error handling for invalid expressions
- Responsive design with gradient backgrounds and hover effects
- Keyboard-friendly input via on-screen buttons

## Technologies Used
- **HTML5**: Structure and semantic markup
- **CSS3**: Styling with gradients, shadows, and transitions
- **JavaScript**: Logic for calculations and DOM manipulation

## How to Use
1. Clone or download the repository
2. Open `index.html` in a web browser
3. Click the buttons to perform calculations
4. Use the clear button to reset or backspace to correct input

## Project Structure
- `index.html`: Main HTML file with calculator layout
- `style.css`: CSS styles for visual design and responsiveness
- `script.js`: JavaScript functions for calculator logic
- `calculator.png`: Icon file for the browser tab
